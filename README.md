# 💰 Finance Management API

RESTful API для комплексної системи управління особистими фінансами. Проєкт дозволяє користувачам вести облік доходів, витрат, здійснювати перекази між рахунками та контролювати свої бюджети.

## ✨ Основні можливості (Features)

- **🔐 Безпека та Ізоляція:** Реєстрація та авторизація користувачів за допомогою JWT. Дані кожного користувача надійно ізольовані.
- **💳 Багатокористувацькі рахунки:** Підтримка кількох рахунків для одного користувача (наприклад: Готівка, Зарплатна картка, Криптогаманець).
- **🔄 Розумні транзакції:** Підтримка трьох типів операцій: `income` (дохід), `expense` (витрата) та `transfer` (переказ між власними рахунками).
- **🏷 Гнучка категоризація:** Можливість створювати власні категорії та додавати масиви тегів до кожної транзакції для глибокої аналітики.
- **📈 Фінансове планування:** Встановлення місячних лімітів (бюджетів) на категорії та створення цілей (скарбничок).

## 🛠 Технологічний стек

- **Серверне середовище:** Node.js
- **Фреймворк:** Express.js
- **База даних:** MongoDB
- **ORM / ODM:** Mongoose для MongoDB
- **Аутентифікація:** JSON Web Tokens (JWT) + bcrypt для хешування паролів
- **Валідація даних:** Joi / Zod (перевірка вхідних даних)

## 🗄 Архітектура бази даних

Проєкт використовує документо-орієнтовану структуру, розроблену для швидкого доступу та масштабування.
